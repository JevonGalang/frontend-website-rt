# 🌐 Bab 10: Real-time Live Sync dengan Socket.io

Sistem backend kini dilengkapi dengan integrasi WebSocket menggunakan **Socket.io** untuk melakukan sinkronisasi data secara real-time pada dashboard panel warga, RT, sekretaris, maupun bendahara tanpa perlu reload halaman secara manual.

---

## 🧐 Mekanisme Kerja Live Sync

1. Client (Frontend) membuat koneksi WebSocket tunggal ke server.
2. Ketika ada aksi perubahan data di backend (misal pembayaran IPL disetujui, warga baru didaftarkan, dll.), server secara otomatis akan memancarkan (broadcast) event `"sync"` dengan kategori tipe data yang berubah.
3. Client menangkap event `"sync"`, memeriksa properti `type`-nya, lalu memicu fetch ulang API (refetch) yang bersangkutan untuk memperbarui UI seketika.

---

## 📡 Daftar Kategori Event `"sync"`

Ketika event `"sync"` terpancar, backend mengirimkan payload objek `{ type: string }`. Berikut adalah daftar tipe event yang dipancarkan beserta pemicunya:

| Nilai `type` | Pemicu Broadcast di Backend | Halaman / UI Frontend yang Harus Refetch |
|---|---|---|
| `"finance"` | Pembayaran IPL/Kas diunggah, pembayaran disetujui/ditolak, pencatatan pengeluaran atau pemasukan baru. | Keuangan, Laporan Kas, Histori Pembayaran Warga, Tracking Tunggakan. |
| `"warga"` | Input data warga baru, edit data warga, verifikasi pendaftaran mandiri warga baru disetujui/ditolak. | Manajemen Warga, List Rumah, List Warga RT, Status Verifikasi Warga. |
| `"pengaduan"` | Pengaduan baru diajukan oleh warga, status pengaduan diperbarui oleh RT/Sekretaris. | Daftar Pengaduan Warga, Panel Tinjauan Pengaduan RT. |
| `"pengajuan"` | Surat pengajuan baru dikirim oleh warga, status pengajuan disetujui/ditolak oleh RT/Sekretaris. | Daftar Pengajuan Warga, Panel Tinjauan Surat RT. |
| `"announcement"` | Pengumuman baru dibuat, diperbarui, atau dihapus oleh pengurus. | Info Landing Page, Widget Pengumuman di Dashboard Warga/Admin. |
| `"agenda"` | Agenda kegiatan baru ditambahkan, diubah, atau dihapus oleh pengurus. | Kalender Kegiatan Warga/RT, List Agenda. |
| `"vote"` | Warga memasukkan suara (cast vote) untuk pemilihan karyawan terbaik. | Halaman Hasil Voting Karyawan. |

## 💻 Panduan Integrasi di Frontend (React / Vue / Vanilla JS)

Berikut contoh kode integrasi di frontend menggunakan pustaka `socket.io-client`:

### 1. Instalasi Pustaka Client
```bash
npm install socket.io-client
```

### 2. Kode Koneksi & Pendengar Event (React/Hooks Example)
```javascript
import { useEffect } from 'react';
import { io } from 'socket.io-client';

const SOCKET_SERVER_URL = "http://localhost:3333";

function useLiveSync(onSyncTrigger) {
    useEffect(() => {
        // Inisialisasi koneksi socket
        const socket = io(SOCKET_SERVER_URL, {
            transports: ['websocket'], // Direkomendasikan menggunakan websocket langsung
            reconnectionAttempts: 5
        });

        socket.on('connect', () => {
            console.log('🌐 terhubung ke real-time socket server backend!');
        });

        // Mendengarkan event sync dari backend
        socket.on('sync', (data) => {
            console.log(`⚡ Menerima request sinkronisasi untuk: ${data.type}`);
            // Panggil callback untuk me-refetch API yang bersangkutan
            if (onSyncTrigger) {
                onSyncTrigger(data.type);
            }
        });

        socket.on('disconnect', () => {
            console.log('❌ koneksi real-time socket terputus!');
        });

        // Bersihkan koneksi saat komponen unmount
        return () => {
            socket.disconnect();
        };
    }, [onSyncTrigger]);
}

export default useLiveSync;
```

### 3. Penggunaan di Dashboard Component (Contoh Refetch Data)
```javascript
import React, { useState, useEffect } from 'react';
import useLiveSync from './useLiveSync';

function FinanceDashboard() {
    const [payments, setPayments] = useState([]);

    const fetchPayments = async () => {
        const res = await fetch("http://localhost:3333/resident/my-payments", { ... });
        const data = await res.json();
        setPayments(data);
    };

    useEffect(() => {
        fetchPayments();
    }, []);

    // Panggil hook live sync
    useLiveSync((type) => {
        if (type === 'finance') {
            console.log('🔄 Memperbarui data keuangan secara real-time...');
            fetchPayments();
        }
    });

    return (
        <div>
            {/* Tampilkan UI data keuangan */}
        </div>
    );
}
```

---

Dengan mekanisme di atas, frontend tidak perlu lagi melakukan polling berkala ke API backend yang membebani server. Sinkronisasi data di layar admin dan warga dijamin **instan**!

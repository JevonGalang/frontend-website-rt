# 📅 Bab 9: Fitur & Endpoint Agenda Kegiatan RT

Bab ini membahas fitur Agenda Kegiatan RT yang memungkinkan RT dan Sekretaris untuk melakukan CRUD (Create, Read, Update, Delete) agenda, serta Warga untuk melihat dan mencari agenda kegiatan di lingkungan RT.

---

## 🗄️ Struktur Tabel Database (`agenda`)

Tabel `agenda` memiliki struktur sebagai berikut:
- **`id`**: INT (Primary Key, Auto Increment)
- **`kategori`**: VARCHAR(100) (Contoh: `KEGIATAN RT`, `KERJA BAKTI`, `RAPAT WARGA`, `KESEHATAN`)
- **`judul`**: VARCHAR(255) (Contoh: `Kerja Bakti & Fogging Nyamuk DBD`)
- **`deskripsi`**: TEXT (Keterangan detail agenda)
- **`tanggal`**: DATE (Tanggal kegiatan dilaksanakan, format: `YYYY-MM-DD`)
- **`waktu`**: VARCHAR(100) (Format bebas, contoh: `07:00 - 11:00 WIB` atau `19:30 WIB - Selesai`)
- **`tempat`**: VARCHAR(255) (Tempat pelaksanaan, contoh: `Balai Warga`, `Pos Satpam`)

---

## 🪙 Endpoint Khusus RT & Sekretaris (`/admin`)

### A. Tambah Agenda Baru
Membuat agenda kegiatan baru di lingkungan RT.

* **Method & Route:** `POST /admin/agenda`
* **Headers:** `Authorization: Bearer <token_jwt_rt_atau_sekretaris>`
* **Request Body:**
  ```json
  {
    "kategori": "KERJA BAKTI",
    "judul": "Kerja Bakti & Fogging Nyamuk DBD",
    "deskripsi": "Kegiatan gotong royong membersihkan saluran air tersumbat serta pelaksanaan pengasapan (fogging nyamuk).",
    "tanggal": "2026-07-12",
    "waktu": "07:00 - 11:00 WIB",
    "tempat": "Area Fasos, Fasum, RT 02"
  }
  ```
* **Response Sukses (200 OK):**
  ```json
  {
    "response": 200,
    "output": {
      "fieldCount": 0,
      "affectedRows": 1,
      "insertId": 1,
      "info": "",
      "serverStatus": 2,
      "warningStatus": 0,
      "changedRows": 0
    },
    "message": "agenda kegiatan berhasil dibuat masbro"
  }
  ```

---

### B. List & Pencarian Agenda
RT/Sekretaris melihat daftar agenda kegiatan. Opsional pencarian via query string `search`.

* **Method & Route:** `GET /admin/agenda` atau `GET /admin/agenda?search=kerja`
* **Headers:** `Authorization: Bearer <token_jwt_rt_atau_sekretaris>`
* **Response Sukses (200 OK):**
  ```json
  [
    {
      "id": 1,
      "kategori": "KERJA BAKTI",
      "judul": "Kerja Bakti & Fogging Nyamuk DBD",
      "deskripsi": "Kegiatan gotong royong membersihkan saluran air tersumbat...",
      "tanggal": "2026-07-12T00:00:00.000Z",
      "waktu": "07:00 - 11:00 WIB",
      "tempat": "Area Fasos, Fasum, RT 02"
    }
  ]
  ```

---

### C. Update Agenda (Partial Update)
Memperbarui rincian agenda berdasarkan ID. Cukup kirim field yang ingin diubah.

* **Method & Route:** `PATCH /admin/agenda/:id`
* **Headers:** `Authorization: Bearer <token_jwt_rt_atau_sekretaris>`
* **Request Params:** `:id` (ID agenda)
* **Request Body:**
  ```json
  {
    "judul": "Kerja Bakti & Fogging Nyamuk DBD (Revisi)",
    "waktu": "08:00 - 12:00 WIB"
  }
  ```
* **Response Sukses (200 OK):**
  ```json
  {
    "response": 200,
    "output": {
      "fieldCount": 0,
      "affectedRows": 1,
      "info": "Rows matched: 1  Changed: 1  Warnings: 0",
      "serverStatus": 2,
      "warningStatus": 0,
      "changedRows": 1
    },
    "message": "agenda kegiatan berhasil diperbarui masbro"
  }
  ```

---

### D. Hapus Agenda
Menghapus agenda dari database berdasarkan ID.

* **Method & Route:** `DELETE /admin/agenda/:id`
* **Headers:** `Authorization: Bearer <token_jwt_rt_atau_sekretaris>`
* **Request Params:** `:id` (ID agenda)
* **Response Sukses (200 OK):**
  ```json
  {
    "response": 200,
    "output": {
      "fieldCount": 0,
      "affectedRows": 1,
      "info": "",
      "serverStatus": 2,
      "warningStatus": 0,
      "changedRows": 0
    },
    "message": "agenda kegiatan berhasil dihapus masbro"
  }
  ```

## 🏠 Endpoint Khusus Warga (`/resident`)

### A. List & Pencarian Agenda Warga
Warga melihat riwayat/daftar agenda kegiatan RT yang aktif. Bisa dicari berdasarkan judul, kategori, atau tempat.

* **Method & Route:** `GET /resident/agenda` atau `GET /resident/agenda?search=Layanan`
* **Headers:** `Authorization: Bearer <token_jwt_warga>`
* **Response Sukses (200 OK):**
  ```json
  [
    {
      "id": 2,
      "kategori": "KESEHATAN",
      "judul": "Layanan Posyandu Balita & Senam Lansia",
      "deskripsi": "Pemberian imunisasi dasar anak, penimbangan berat badan balita, penyuluhan gizi...",
      "tanggal": "2026-08-05T00:00:00.000Z",
      "waktu": "08:00 - 11:30 WIB",
      "tempat": "Pos Satpam Utama SGP"
    }
  ]
  ```
